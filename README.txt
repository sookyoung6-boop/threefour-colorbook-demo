ThreeFour Coloring Demo · Package
==================================

폴더 구조 (이대로 유지하세요):
  threefour_demo/
    ├─ threefour_demo.html       ← Safari로 이 파일 열기
    └─ audio/
       └─ p02_wise_lady.mp3      ← Page 02 오디오

사용법
------
1. 이 zip을 풀어서 폴더 그대로 보관
2. threefour_demo.html을 Safari로 열기
3. (선택) Safari → 공유 → '홈 화면에 추가' 하면 풀스크린 시연 모드

오디오 추가하기
--------------
audio/ 폴더에 다음 파일명으로 mp3를 두면 자동 인식:
  p02_wise_lady.mp3   (이미 포함)
  p03_xxx.mp3
  p04_xxx.mp3
  ...
  p10_xxx.mp3

새 트랙 파일명만 HTML 코드의 AUDIO_TRACKS 부분에서 수정하면 됨.
